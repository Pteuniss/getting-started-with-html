Notes about the internet

How the web works

The web is a bunch of computers working together. When someone uses a browser and opens up a webpage, the computer sends a http request to a server. Servers are big computers on which data is stored. There are millions of servers around the world. After the request is recognized, it sends the information to the webbrowser and the browser shows the user the requested information on his or her screen.

HTML

HTML stands for HyperText Markup Language. These texts are documents on the internet and they form the most content on the web. Beside HTML you have pictures, video and many more things! HTML documents contain text, which describe what the user sees.
